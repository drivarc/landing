# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]
### Added
- İçerik Güvenlik Politikası (CSP) koruması eklendi.
- Service Worker ile çevrimdışı PWA desteği devreye alındı.
- Görseller için `loading="lazy"` (Lazy load) optimizasyonu eklendi.
- Çerez kullanım onayı popup'ı (Consent mekanizması) dahil edildi.
- GitHub Actions CI pipeline yapılandırıldı.

## [1.0.0] - 2026-04-14
### Added
- Ana sayfa (`index.html`), iletişim, gizlilik (`privacy.html`), kullanım şartları (`terms.html`) ve 404 sayfaları eklendi.
- `app.js` ve `styles.css` ile site frontend mantığı ve tasarımlar tamamlandı.
- Çoklu dil (en, de, ru, ar, zh) destekleyici klasörler eklendi.
- Temel SEO ayarları (`robots.txt`, `sitemap.xml`, `seo-jsonld.js`) yapılandırıldı.
